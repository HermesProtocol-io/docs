Welcome to the documentation of Hermes Protocol.

Hermes Protocol is a communication platform 100% free for everyone, allowing users to receive instant alerts about on-chain events, such as wallet transactions (whale monitoring, for ex.), NFT purchases/bids/lists/etc., price alerts, governance polls, validator profile changes (commission %, name, etc.), & more.

:::info

**Supported Blockchains**: Terra, Terra Classic. _(up-to-date list can be found [here](https://github.com/HermesProtocol-io/assets/tree/main/static/blockchains))_

:::
