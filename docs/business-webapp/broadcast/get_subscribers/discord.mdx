To obtain subscribers, it's essential to have the Hermes Protocol bot integrated into your server. Add the bot to your server with this [link](https://hermesprotocol.io/r/discord-bot-invite), and follow the on-screen instructions.



![create list](/img/broadcast/discord-auth.png)

:::info
You can only invite the bot to a server where you have Manage Server permissions.
:::

Once the bot is successfully added to your server, use the following command to generate the subscribe List embed:

![discord command](/img/broadcast/discord-comand.png)
