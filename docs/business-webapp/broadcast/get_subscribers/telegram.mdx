To obtain subscribers, it's essential to have the Hermes Protocol bot integrated into your server. Start the connection by clicking the [link](https://hermesprotocol.io/r/telegram-bot-invite), and follow the on-screen instructions.

![create list](/img/broadcast/telegram-auth.png)

:::info
You can only invite the bot to a server where you have Manage Server permissions.
:::

Once the bot is successfully added to your server, use the following command to generate the subscribe List embed:
