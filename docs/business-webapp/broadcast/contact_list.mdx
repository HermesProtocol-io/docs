 "Contact Lists" are collections of Subscribers that Organizations can create and manage within the platform. These lists facilitate targeted, personalized communications. They allow Organizations to efficiently deliver tailored notifications or messages to the right audience, enhancing user experience and engagement.

![main screen](/img/broadcast/main_screen.png)

To begin, simply click the "+" icon located at the top right corner of the screen.

![create list](/img/broadcast/create_screen.png)

| Field                | Description                                                                     |                                                                                                   | 
| :------------------ | :------------------------------------------------------------------------------ |
| Internal            | Internal Name used to facilitate management of the lists                                 |
| Public              | Public Name that appears to users when subscribing for certain Communications |
| Short Description   | Description shown to the user on what he is subscribing to                |

After creating your first Contact List you can start collecting subscribers in your [Discord](get_subscribers/discord) and [Telegram](get_subscribers/telegram) (or your website/dApp using our SDK or API).
