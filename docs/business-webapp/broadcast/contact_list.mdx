 "Contact Lists" are collections of user contacts that businesses can create and manage within the platform.
  These lists facilitate targeted, personalized communications by grouping users based on specific attributes or criteria. They allow businesses to efficiently deliver tailored notifications or messages to the right audience, enhancing user experience and engagement.

![main screen](/img/broadcast/main_screen.png)

To begin, simply click the "+" icon located at the top right corner of the screen.

![create list](/img/broadcast/create_screen.png)

| Field                | Description                                                                     |                                                                                                   | 
 | :------------------ | :------------------------------------------------------------------------------ |
 | Internal            |  Name used for internal Management of the lists                                 |
 | Public              | Name that will show up for the user when subscribing for certain communications |
 | Short Description   | Description that will show to the user what he is subscribing to                |

 
 After creating your first list you can start collecting subscribers in your [Discord](get_subscribers/discord) and [Telegram](get_subscribers/telegram)
