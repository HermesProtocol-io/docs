### Request


To create a Wallet Monitor notification using the Hermes API, construct the request body according to the [general structure](/public-api/notification). For the `notificationParams` section, adhere to the Body structure outlined below:

**Body Structure**

 | Parameter                               | Requirement | Type    | Description                                        |
 | :-------------------------------------- | :---------- | :------ | :------------------------------------------------- | 
 | notificationParams.walletAddress        | mandatory   | String  | Address that the user wants to watch               |
