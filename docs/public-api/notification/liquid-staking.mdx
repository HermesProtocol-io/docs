### Request

**Body Structure**

 | Parameter                               | Requirement | Type    | Description                                                                                         |
 | :-------------------------------------- | :---------- | :------ | :-------------------------------------------------------------------------------------------------- | 
 | notificationParams.contract             | mandatory   | String  | Params of the notification                                                                          |
 | notificationParams.tokenContract        | mandatory   | String  | Params of the notification                                                                          |
 | notificationParams.stakes.flag          | mandatory   | Boolean | if the observation is made about staking movements in the blockchain                                |
 | notificationParams.stakes.value         | optional    | Float   | value from when the staking is taken in consideration                                               |
 | notificationParams.stakes.type          | optional    | String  | use [priceType](/public-api/ids) varible value from when the staking is taken in consideration      | 
 | notificationParams.queueUnbond.flag     | mandatory   | Boolean | if the observation is made about queueUnbond movements in the blockchain                            |
 | notificationParams.queueUnbond.value    | optional    | Float   | value from when the queueUnbond is taken in consideration                                           |
 | notificationParams.queueUnbond.type     | optional    | String  | use [priceType](/public-api/ids) varible value from when the queueUnbond is taken in consideration  |
 | notificationParams.submitBatch.flag     | mandatory   | Boolean | if the observation is made about submitBatch movements in the blockchain                            |
 | notificationParams.submitBatch.value    | optional    | Float   | value from when the submitBatch is taken in consideration                                           |
 | notificationParams.submitBatch.type     | optional    | String  | use [priceType](/public-api/ids) varible value from when the submitBatch is taken in consideration  |
 | notificationParams.finishUnbond.flag    | mandatory   | Boolean | if the observation is made about finishUnbond movements in the blockchain                           |
 | notificationParams.finishUnbond.value   | optional    | Float   | value from when the finishUnbond is taken in consideration                                          |
 | notificationParams.finishUnbond.type    | optional    | String  | use [priceType](/public-api/ids) varible value from when the finishUnbond is taken in consideration |
 | notificationParams.scope                | mandatory   | String  | use [scope](/public-api/ids) varible value                                                          |


:::info

In this section we only show the notification parameters  specific for liquid staking. To see the rest of the body structure please check the [notification](/public-api/notification) 

:::

**stakes / queueUnbond / submitBatch / finishUnbond**

| Param | Type       | Description                                                                          |
| :---- | :--------- | :----------------------------------------------------------------------------------- |
| flag  | Boolean    | Whether the user want receive alert about this event or not                          |
| value | Float      | Value used on the event                                                              |
| type  | String     | Type of comparison used between the value configured and the value used on the event |

Example

```json
{
  "stakes": {
    "flag": true,
    "value": 10,
    "type": "="
  },
  "queueUnbond": {
    "flag": true,
    "value": 25,
    "type": "<"
  },
  "submitBatch": {
    "flag": true,
    "value": 12,
    "type": "<="
  },
  "finishUnbond": {
    "flag": true,
    "value": 100,
    "type": ">"
  }
}
```
