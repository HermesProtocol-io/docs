### Scope

| scope    | Scope Type      | Description                                        | Notification Type                                     |
| :------- | :-------------- | :------------------------------------------------- | :---------------------------------------------------- |
| zc5_0001 | My Wallet       | Scope is the wallet of the user                    | NFT Monitor, Profile Changes, Staking, Liquid Staking |
| zc5_0002 | Unique          | Scope is an unique contract: NFT or Validator      | NFT Monitor, Profile Changes                          |
| zc5_0003 | Other Wallet    | Scope is the wallet that do not belong to the user | Staking, Liquid Staking                               |
| zc5_0004 | My Domain       | Scope is the domain registered in the user wallet  | Domain Changes, Domain Registration                   |
| zc5_0005 | Other Domain    | Scope is a domain not associated to the user       | Domain Changes, Domain Registration                   |
