This command allows you to delete one your previously created notifications

```
    /notification delete
```

## NFT

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/notification delete nft
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**channel\*:**

    type: Text Channel
    description: Channel where the alert are being sent.

**collection\*:**

    type: Options
    description: NFT collection address to unsubscribe.
    choices: [ Your NFT collections addresses ]

**Example Usage**

```
/notification delete nft channel:#channel collection:terra1...
```

## Governance

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/notification delete governance
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**channel\*:**

    type: Text Channel
    description: Channel where the polls are being sent to.

**address\*:**

    type: String
    description: Inserts the smart contract where the polls are created.

:::note
The only governance polls that are accepted must follow the standard used by Anchor Protocol and Mirror Protocol.
:::

Usage example:

```
/notification delete governance blockchain:Terra channel:#governance address:terra1...
```
