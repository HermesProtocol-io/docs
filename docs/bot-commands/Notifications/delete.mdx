This command allows you to delete one your previously created notifications

```
    /notification delete
```

## NFT

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/notification delete nft
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_\*

**id\*:**

    type: Options
    description: Allows you to delete one of the previously created NFT Feeds/Monitor.
    choices: [ Your created alerts ]

**Example Usage**

```
/notification delete nft id:
```
