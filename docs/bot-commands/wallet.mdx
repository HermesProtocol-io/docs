## Watch

This command let you configure a wallet to watch and will alert you via your DMs (Direct Messages).

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/wallet watch
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**address\*:**

    type: String
    description: Wallet address to observe.

**blockchain\*:**

    type: Options
    description: Blockchain where that address exists.
    choices: [ Terra Classic, Terra ]

**Example Usage**

```
/wallet watch address:terra1... blockchain:Terra
```

## Unwatch

This command let you remove a wallet from your watch list.

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/wallet unwatch
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**address\*:**

    type: String
    description: Wallet address to remove from your watch list.

**Example Usage**

```
/wallet unwatch address:terra1...
```
