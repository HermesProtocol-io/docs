These commands are used to watch all transactions from one or multiple wallets.

```
/wallet
```

## Watch

This command let you configure a wallet to watch and will alert you via your DMs (Direct Messages).

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/wallet watch
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**address\*:**

    type: String
    description: Wallet address to observe.

**blockchain\*:**

    type: Options
    description: Blockchain where that address exists.
    choices: [ Terra Classic, Terra ]

**Example Usage**

```
/wallet watch address:terra1... blockchain:Terra
```

</TabItem>

<TabItem value="telegram" label="Telegram">
<p><button class={styles.telegram}>Watch Wallet</button></p>

To create a Wallet Watch alert you need to follow these steps:

1. Press the Watch Wallet button;
2. Insert the Wallet address you wish to watch;
3. Choose the blockchain where you want to track its transactions.

![][telegram-ww]

Nice job! You can now receive alerts when that wallet is used.

## List

This command exhibits a list with all the wallets that you are currently watching.

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/wallet list
```

</TabItem>

<TabItem value="telegram" label="Telegram">
<p><button class={styles.telegram}>List Wallet</button></p>

To list all the wallets you are currently tracking you only need to press the List Wallet button and you will receive a list with all the wallets.

![][telegram-wl]

Well done! You have listed all your wallets.

## Unwatch

This command let you remove a wallet from your watch list.

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/wallet unwatch
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**address\*:**

    type: String
    description: Wallet address to remove from your watch list.

**Example Usage**

```
/wallet unwatch address:terra1...
```

</TabItem>

<TabItem value="telegram" label="Telegram">
<p><button class={styles.telegram}>Unwatch Wallet</button></p>

To remove a Wallet Watch alert you need to follow these steps:

1. Press the Unwatch Wallet;
2. Select the Wallet you wish to remove.

![][telegram-uw]

Well done! You have successfully removed a wallet watch.

</TabItem>
