This command allows you to check all the information of a specific wallet in any of the supported blockchains.

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/get wallet
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**address\*:**

    type: String
    description: Wallet address to get information from.

**blockchain\*:**

    type: Options
    description: Blockchain where that address exists.
    choices: [ Terra Classic, Terra ]

**visibility:**

    type: Options
    description: Choose whenever the message the bot sends is public (channel wide) or private (your eyes only).

**Example Usage**

```
/get wallet address:terra...blockchain:Terra
```
