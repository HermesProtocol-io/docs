## Create Price

This command allows you to setup a price alert, that will notify you when the coin you want reaches the price you pretend.

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/alert create price
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**blockchain\*:**

    type: Options
    description: The blockchain where the coin exist to.
    choices: [ Terra Classic, Terra ]

**token\*:**

    type: String
    description: The cryptocurrency you wish to create the price alert for.

**value\*:**

    type: String
    description: The price you wish that the coin reaches in order to be notified.

**type\*:**

    type: Options
    description: Define on what condition relative to the value you will be notified
    choices: [ greater than or equal to, greater than, lower than or equal to, lower than, equal to ]

**Example Usage:**

```
/alert create price blockchain:Terra token:axlUSDT / LUNA Pair (Phoenix) price:5 type:greater than or equal to
```

</TabItem>

<TabItem value="telegram" label="Telegram">
<p><button class={styles.telegram}>Subscribe Price Alerts</button></p>

In order to create a price alert you need to follow these steps:

1. Press the Subscribe Price Alerts Button;
2. Select the blockchain you wish;
3. Select the Token pair you desire;
4. insert the price you want the pair to reach in order to be notified.

Well done! you have created a price alert.

## Delete

This command allows you to delete one of your previously created alerts.

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/alert delete
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_\*

**id\*:**

    type: Options
    description: The alerts you previously created.
    choices: [ Your created alerts ]

**Example Usage**

```
/alert delete id: price alert for ...
```

</TabItem>
