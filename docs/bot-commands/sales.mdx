## Subscribe

This command let you subscribe a NFT collection sales.

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/sales subscribe
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**blockchain\*:**

    type: Options
    description: Blockchain where that address exists.
    choices: [ Terra Classic, Terra ]

**collection\*:**

    type: Collection
    description: NFT collection address to subscribe.

**channel:**

    type: Text Channel
    description: Channel to send the notifications.

**Example Usage**

```
/sales subscribe blockchain:Terra collection:terra1... channel: #channel
```

or

```
/sales subscribe collection_address:terra1... blockchain:Terra channel:#channel
```

</TabItem>

<TabItem value="telegram" label="Telegram">
<p><button class={styles.telegram}>Subscribe</button></p>

To subscribe to a NFT collection you need to follow these steps:

1. Press the Subscribe button;
2. Insert the collection address you want to follow;
3. Choose the blockchain where you want to track its transactions.

![][telegram-ns]

Well done! You have subscribed to a collection.

## Unsubscribe

This command allows you to unsubscribe one of yours NFT collections sales previously configured.

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/sales unsubscribe
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**channel\*:**

    type: Text Channel
    description: Channel where the notifications are being sent.

**collection\*:**

    type: Options
    description: NFT collection address to unsubscribe.
    choices: [ Your NFT collections addresses ]

**Example Usage**

```
/sales unsubscribe channel:#channel collection:terra1...
```

</TabItem>

<p><button class={styles.telegram}>Unsubscribe</button></p>

To unsubscribe from a NFT collection you need to follow these steps:

1. Press the Unsubscribe button;
2. Choose the collection you want to unsubscribe from.

Well done! You have unsubscribe from a collection
