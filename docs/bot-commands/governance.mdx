This command is used to subscribe to all the governance polls that are created

```
/governance
```

## Subscribe

This command allows you you to create governance poll feed, where all the created polls are posted with a dedicated thread where users will be able to discuss.

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/governance subscribe
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**blockchain\*:**

    type: Options
    description: Blockchain where the poll exists.
    choices: [ Terra Classic, Terra ]

**channel\*:**

    type: Text Channel
    description: Channel where the polls will be sent to.

**address\*:**

    type: String
    description: Inserts the smart contract where the polls exists.

:::note
In order for a governance polls to be accepted they must follow the standard used by Anchor Protocol and Mirror Protocol.
:::

**url:**

    type: link
    description: Insert the protocol website where all the proposals https://protocol.io/gov/%id%.

**mention:**

    type : Role
    description: Insert which role you wish to mention whenever a new poll is posted.

**Usage example**

```
/governance subscribe blockchain:Terra channel:#governance address:terra1... url:https://protocol.io/gov/%id% mention:@role
```

</TabItem>

<TabItem value="telegram" label="Telegram">
<p><button class={styles.telegram}>Governance Subscribe</button></p>

To subscribe to a governance polls you need to follow these steps:

1. Press the Governance Subscribe button;
2. Insert the Smart Contract you wish to track;
3. Select the blockchain where the polls are created.

Well done! You have subscribed to every governance poll that is created with that smart contract.

## Unsubscribe

The Governance unsubscribe command allows you to stop receiving the polls on the channel you chose.

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/governance unsubscribe
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**channel\*:**

    type: Text Channel
    description: Channel where the polls are being sent to.

**address\*:**

    type: String
    description: Inserts the smart contract where the polls are created.

:::note
The only governance polls that are accepted must follow the standard used by Anchor Protocol and Mirror Protocol.
:::

Usage example:

```
/governance unsubscribe blockchain:Terra channel:#governance address:terra1...
```

</TabItem>

<TabItem value="telegram" label="Telegram">
<p><button class={styles.telegram}>Governance Unsubscribe</button></p>

To unsubscribe from a governance poll you need to follow these steps:

1. Press the Governance Unsubscribe;
2. Select the contract you wish to unsubscribe from.

Well done! You have unsubscribed from a governance poll.

</TabItem>
