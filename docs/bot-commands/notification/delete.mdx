This command provides the ability to delete a notification that you have previously created. Simply select the notification you wish to delete, and it will be removed from your account.

```
/notification delete
```

## NFT

<Tabs groupId="platforms">
<TabItem value="discord" label="Discord" default>

```
/notification delete nft
```

<p>
  <strong>Parameters:</strong>
</p>

\*_Required_

**id\***

    type: Option
    description: Your previously created NFT notifications.
    choices: [ Your notifications ]

**Example Usage**

```
/notification delete nft id:NFT Monitor for Skeleton Punks (Skeleton Punk #878) on Terra (DMs)
```
